1. A CHRISTIAN Memorandum.

    _ CHAP. 1.

      * Three arguments to moue vs to clemency.

      * First Argument the guilt of Sinne.

      * Second argument is charity towards the ſinner.

      * Second argument is charity towards the ſinner.

      * Second argument is charity towards the ſinner.
